# Fibonacci LCD Pendant - RoundyFi

Wearable pendant made from a [RoundyFi](https://github.com/sbcshop/RoundyFi) 1.28-inch 240x240 LCD display with built-in ESP-12E ESP8266 microcontroller with wi-fi.
